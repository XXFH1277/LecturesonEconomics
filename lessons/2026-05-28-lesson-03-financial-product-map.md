# 第三课：金融产品地图：所有权、债权债务、信托关系和衍生工具

## 基本信息

- 日期：2026-05-28
- 数据截至：北京时间 2026-05-28
- 主题：把股票、债券、基金、外汇、商品和衍生工具放进一张关系地图
- 学习目标：不再把金融产品当成散乱名词，而是先看它背后代表什么关系
- 适合阶段：零基础入门后
- 输出定位：教学与研究分析，不构成具体买卖建议

## 本课怎么学

这一课不要急着记产品细节。先记住一个问题：

> 这个金融产品背后，到底是谁和谁发生了什么关系？

本课分三层掌握：

1. 必须掌握：所有权、债权债务、信托/受托管理、货币交换、衍生合约。
2. 需要理解：一级市场和二级市场的区别，基金为什么不是衍生品。
3. 先认识即可：期货、期权、互换、保证金、套期保值。这些后续会单独讲。

可回顾第 1 课：资产类别、价格、风险、收益、流动性。  
可回顾第 2 课：制度、监管、政策如何影响市场规则。

## 第一大板块：理论基础

### 1. 为什么要按“关系”分类

很多新手一听股票、债券、基金、外汇、期货、期权，就觉得全是陌生词。其实可以先不用背名字，而是看关系。

金融产品大致可以问五个问题：

1. 我买了以后，拥有了什么？
2. 谁欠谁钱？
3. 谁替谁管理钱？
4. 谁和谁交换货币？
5. 这是不是一份关于未来价格或现金流的合约？

这五个问题对应五类关系：

```text
所有权关系 -> 股票、部分商品现货
债权债务关系 -> 债券、存款
信托/受托管理关系 -> 基金、部分信托
货币交换关系 -> 外汇
衍生合约关系 -> 远期、期货、期权、互换
```

停一下：如果你买一只基金，你是直接拥有基金里每一家公司的股票，还是拥有基金份额？

### 2. 所有权关系：我拥有一部分资产

所有权（Ownership）是最直观的一类关系。你买了某个东西，就拥有它的一部分或全部。

股票是典型例子。买股票，本质上是买公司的一小部分所有权。公司经营得好，未来可能有利润增长、分红或估值提升；公司经营得差，股价也可能下跌。

商品现货也可以是所有权关系。例如持有实物黄金，就是拥有黄金本身。但如果你买的是黄金 ETF、黄金期货或黄金矿业公司股票，关系就不完全一样。

常见误区：

- 误区：股票就是一张可以随时退给公司的票。
- 更准确：上市后日常买卖大多发生在二级市场，是投资者之间转让已有股票。

可回顾第 1 课：价格不是价值，价格是市场此刻愿意成交的数字。

### 3. 一级市场和二级市场：钱流向不一样

一级市场（Primary Market）是发行市场，重点是公司或政府第一次把证券卖给投资者来融资。

二级市场（Secondary Market）是流通市场，重点是投资者之间互相买卖已经发行的证券。

用股票举例：

```text
IPO 或增发
  -> 公司发行股票
  -> 投资者把钱给公司
  -> 公司获得融资

交易所日常买卖
  -> 投资者 A 卖给投资者 B
  -> 公司通常不直接收到这笔交易的钱
```

这点非常重要。很多人以为“我买了股票，钱就进了公司”。如果是二级市场买入，通常不是这样。

停一下：如果你在交易所买入某上市公司股票，这笔钱多数情况下是给了公司，还是给了卖给你股票的投资者？

### 4. 债权债务关系：我借钱给别人

债权债务关系（Debt-Credit Relationship）就是一方借钱给另一方。

债券（Bond）是典型例子。政府或企业发行债券，意思是向投资者借钱，并承诺按规则付利息、到期还本金。

常见债券：

- 国债：国家发行，通常信用等级较高。
- 地方债：地方政府或相关主体发行。
- 公司债/企业债：企业发行，信用风险通常高于国债。

为什么公司债收益率通常要比国债高？因为投资者承担了更高违约风险，需要风险补偿。

可回顾第 1 课：收益不是白来的，高收益通常要问风险在哪里。

### 5. 信托或受托管理关系：我把钱交给别人按规则管理

信托（Trust）或受托管理（Fiduciary / Entrusted Management）关系的核心是：

```text
有人出钱
  -> 有人按约定管理钱
  -> 有人获得收益或承担结果
```

基金就是非常典型的集合投资形式。投资者把钱放进基金，基金按合同、招募说明书或指数规则投资股票、债券、黄金、现金等资产。

基金里常见三个角色：

- 投资者：出钱的人。
- 管理人：管理基金的人或机构。
- 托管人：保管基金资产、监督资金安全的机构。

基金不是衍生品。虽然基金净值会随着股票、债券、黄金、原油等底层资产变化，但你买的是基金份额，不是一份约定未来价格结算的合约。

停一下：基金的价值会随持仓变化，这一点和衍生品“联动基础资产”有点像。但为什么基金通常仍不算衍生品？

### 6. 货币交换关系：外汇不是单纯换钱

外汇（Foreign Exchange / FX）是不同货币之间的交换。

最表层看，它是价格：

```text
人民币兑美元汇率
  -> 用多少人民币换 1 美元
```

但第 2 课已经讲过，外汇不只是价格问题，还涉及跨境资本流动、贸易、监管、金融安全和国际收支。

可回顾第 2 课：外汇为什么不是简单的自由买卖。

### 7. 衍生合约关系：围绕未来价格或现金流的合约

衍生工具（Derivative）不是凭空来的，它的价值通常来自某个基础资产或基础变量。

基础资产可以是：

- 股票或股票指数。
- 债券或利率。
- 黄金、原油、农产品。
- 汇率。
- 信用风险。

常见衍生工具：

- 远期（Forward）：双方私下约定未来某个价格交易。
- 期货（Futures）：交易所标准化的未来合约。
- 期权（Options）：买方付权利金，获得未来按约定价格买入或卖出的权利。
- 互换（Swaps）：双方交换未来现金流，例如固定利率换浮动利率。

衍生工具常有几个特征：

- 和基础资产或变量联动。
- 是关于未来的合约。
- 可能有杠杆，放大收益和亏损。
- 可以投机，也可以套期保值。

套期保值（Hedging）不是为了赌方向，而是为了降低不确定性。航空公司用油价相关工具锁定成本，就是经典直觉案例。

## 第二大板块：实时背景与市场传导

### 1. 监管机构为什么反复做投资者教育

SEC Investor.gov 长期把股票、债券、共同基金、ETF 等作为投资者教育的基础内容；CFTC 也把期货、期权和衍生品风险作为公众教育重点。

这说明一个现实问题：金融产品看起来只是“买什么”，但实际上涉及法律关系、交易机制、费用、流动性、杠杆和监管。

### 2. 为什么基金、ETF、期货不能混在一起学

基金和 ETF 更多是集合投资或受托管理关系。期货和期权更多是围绕未来价格的合约关系。

它们都可能和股票、黄金、原油有关，但关系不同：

```text
黄金 ETF -> 通常是基金份额，底层可能持有黄金或相关资产
黄金期货 -> 未来某个时间按规则结算或交割的合约
黄金矿业股 -> 买的是黄金矿业公司的股权
实物黄金 -> 直接拥有黄金
```

同样是“黄金相关”，风险结构完全不同。

## 政治、制度和规则视角

- 涉及的制度：证券发行、交易所交易、基金监管、期货监管、投资者保护。
- 已确认事实：不同金融产品受不同法律、交易所、监管机构和信息披露要求约束。
- 市场可能如何传导：规则改变会影响融资难度、交易成本、杠杆限制、风险暴露和投资者信心。
- 仍需核验或观察：具体国家和地区规则不同，涉及实际产品时必须查发行文件、监管规定和交易所规则。

## 案例拆解

### 同样和原油相关，产品关系可能完全不同

- 实物原油：商品所有权，但普通个人几乎不会直接持有。
- 能源公司股票：所有权关系，买的是公司一部分股权。
- 能源基金或 ETF：受托管理关系，买的是基金份额。
- 原油期货：衍生合约关系，围绕未来原油价格。

分析推理：同一个主题不等于同一种风险。油价上涨可能利好能源公司利润，但原油期货还受期限结构、保证金、展期等影响。

## 个人情境连接

- 如果你看到一个产品名字里有“黄金”“原油”“AI”“债券”，先不要急着判断好坏。
- 第一步问：它到底是股票、基金、债券、外汇，还是衍生合约？
- 第二步问：收益来自哪里，风险由谁承担？
- 第三步问：它适合学习、观察、长期配置，还是高风险交易？

## 结论边界

- 可以确定：金融产品可以先按关系分类，关系比名字更重要。
- 不能确定：具体产品风险必须看合同、招募说明书、交易所规则和监管文件。
- 需要继续观察：后续学习基金、债券、衍生工具时，要不断回到这张地图。
- 不构成投资建议：本课只讲分类框架，不建议购买任何具体产品。

3. 为什么基金不是衍生品？
4. 同样是黄金相关产品，实物黄金、黄金 ETF、黄金期货、黄金股有什么不同？
5. 套期保值和投机有什么区别？

## 练习题参考思路

1. 股票是所有权，债券是债权债务，基金是受托管理，外汇是货币交换，期货是衍生合约。
2. 二级市场是投资者之间转让已有证券，钱通常给卖方投资者。
3. 基金买的是份额，不是未来结算合约，通常没有期货期权那种保证金和未来履约结构。
4. 先看买的是什么关系，再看价格跟什么变量联动。
5. 套期保值是降低已有风险，投机是主动承担风险换取收益可能。
