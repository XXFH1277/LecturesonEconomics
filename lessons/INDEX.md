# 课程归档索引

每次教学内容都应固定保存为 Markdown 文件，文件名建议使用：

```text
lessons/YYYY-MM-DD-topic-slug.md
```

每篇课程至少包含：

- 日期和数据截至时间。
- 本节主题和学习目标。
- 已核验来源。
- 核心概念。
- 真实案例或实时事件。
- 结论边界。
- 2 到 5 道练习题。
