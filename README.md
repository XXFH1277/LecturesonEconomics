# 实时经济、投资与大宗商品课堂

这是一个持续更新中的中文经济学与投资学习项目，用真实市场、真实公司、基金、黄金、原油、宏观政策和国际事件辅助理解经济学、社会学、心理学与市场行为。

项目地址：https://github.com/XXFH1277/LecturesonEconomics

> 持续更新中：后续每次正式教学都会尽量保存为课程笔记，并把稳定的教学流程沉淀为可复用的 skills，方便自己复习，也方便更多人一起学习。

## 项目内容

- 实时市场与时局雷达：股票、基金、利率、汇率、黄金、原油和重大国际事件。
- 系统课程：经济学、投资研究、市场结构、行为金融、社会心理和个人决策。
- 案例研究：真实公司、股票、基金、ETF、行业、商品和宏观事件。
- 来源核验：涉及实时市场、公司情况、基金持仓、政策、战争、库存、供需和价格的数据，都要求注明出处和数据日期。
- 零基础友好：专业名词、英文和缩写会尽量附中文翻译、白话解释和使用场景。
- 双板块课程：每节课先讲理论基础，再讲实时背景和市场传导。
- 课程归档：每次教学内容固定保存，便于 GitHub 查看、复盘和继续扩展。

## 学习方式

这个项目默认按下面的方式学习：

1. 先读 [学习状态交接](./LEARNING_STATE.md)，确认当前进度和下节课安排。
2. 再看实时市场与时局雷达。
3. 再讲系统知识和核心概念。
4. 用真实公司、基金、商品或事件做案例。
5. 标明事实来源、数据日期和结论边界。
6. 留少量练习题，用来把知识转成判断力。
7. 课程结束后保存本课内容、学习交接和下节课安排。

## 使用原则

- 所有涉及最新公司情况、股票、基金、黄金、原油、政策、战争、制裁、财报、库存、供需、利率、汇率和价格的数据，都要联网核验。
- 来源优先级是：原始披露或官方数据 > 交易所和监管机构 > 权威国际组织和央行 > 基金公司和公司 IR > 权威媒体 > 社区整理或 GitHub 项目。
- 可以用 GitHub 寻找开源数据工具、研究模板和相关项目，但市场事实必须继续追溯到原始来源。
- 输出定位是教学和研究分析，不构成具体买卖建议。

## 适合谁

- 想从零开始系统理解股票、基金、宏观经济和大宗商品的人。
- 想把实时新闻、财报、战争、政策和市场波动放进同一套分析框架的人。
- 想学习如何查来源、读报告、做案例拆解和避免单一因果解释的人。
- 想把个人工作、收入、消费、持仓和风险管理联系起来思考的人。

## 默认学习结构

每次课程建议按下面的顺序组织：

1. 实时雷达：股票、基金、利率、汇率、黄金、原油、重大国际事件。
2. 理论基础：参考教材式顺序，把概念从零开始讲清楚。
3. 实时背景：用真实市场、公司、基金、黄金、原油或政策事件验证概念。
4. 今日主线：选择一个最值得学习的传导链条。
5. 案例拆解：用真实公司、基金、商品或事件验证概念。
6. 结论边界：说明哪些是事实，哪些是推理，哪些需要后续验证。
7. 练习题：2 到 5 道，覆盖理解、判断和个人情境。

## 课程归档规则

每次正式教学后，都应保存到 `lessons/`，并更新 [课程归档索引](./lessons/INDEX.md)。推荐文件名格式：

```text
lessons/YYYY-MM-DD-topic-slug.md
```

如果一节课形成了新的稳定流程，应该同步更新 `skills/`，让这个项目不只积累知识，也积累学习方法。

每节课结尾还必须包含：

- 学习交接：本课学了什么、要复习什么、哪些概念还不稳。
- 下节课安排：下节课主题、目标、建议案例、必须解释的关键词。
- 跨设备续学提示：下次从哪个文件开始读。

## 个人情境输入

后续如果要把课程贴近现实，可以逐步补充：

- 关注清单：股票、基金、ETF、行业、国家、商品。
- 持仓结构：只需要大致类别和比例，不需要透露隐私金额。
- 时间周期：短期观察、长期学习、组合复盘或职业决策。
- 约束条件：现金流、风险承受能力、工作行业、生活成本压力。
